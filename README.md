# 🎬 Movie Booking App
## 📌 Project Overview
### The Movie Booking App is a user-friendly web application that allows users to conveniently book movie tickets, manage their bookings, and download tickets in PDF format. The goal of this project is to provide a seamless and interactive movie booking experience.

## 🚀 Key Features
### 1. Book Movie Tickets – Easily fill in your details to book your movie ticket.

### 2. Edit Booking Details – Modify your ticket information in case of incorrect inputs.

### 3. You can also delete your tickets.

### 4. View Booking History – Access a complete list of all your past bookings.

### 5. Download Ticket as PDF – Instantly download your booked ticket in a well-formatted PDF file.

##  🛠️ Tech Stack

The project is developed using the following technologies:

### Technology	Description
1. Java	Programming language used for backend development
2. Spring Boot	Framework for building the backend REST APIs and handling business logic
3. Thymeleaf	Server-side Java template engine used to build dynamic HTML pages
4. Spring Data JPA	For easy database interaction using repository abstractions
5. MySQL	For storing booking data
6. Maven	Build and dependency management tool
7. iText (PDF Library)	Used to generate PDF tickets after successful booking
8. HTML/CSS	Basic front-end styling and structure within Thymeleaf templates

⚙️ The project follows the MVC (Model-View-Controller) architecture.
